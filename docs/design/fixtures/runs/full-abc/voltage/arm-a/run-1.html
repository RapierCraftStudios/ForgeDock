capabilities
compute

On-demand GPUs

A100s, H100s, L40s. Pick in a decorator argument. Billed per second while your function runs — not per hour, not per reservation.

@voltage.function(gpu="H100")
scale

Zero to thousands

Submit 5,000 tasks. Voltage spins up containers in parallel, fans out work, collects results. No queue configuration. No autoscaling policy YAML.

results = process.map(dataset, concurrency=5000)
inference

Inference endpoints

One decorator converts any function into a persistent HTTP endpoint. Auto-scales to traffic, scales to zero at idle. No Kubernetes.

@voltage.endpoint(min_replicas=0)
scheduling

Cron + job scheduling

Schedule retraining pipelines, data processing jobs, or nightly evals. Trigger from CI, webhooks, or a cron expression — all in Python.

@voltage.cron(schedule="0 2 * * *")
cold start

Sub-200ms cold starts

Image layers are cached at the network edge. Container memory snapshots cut cold start to under 200ms — even for PyTorch environments.
