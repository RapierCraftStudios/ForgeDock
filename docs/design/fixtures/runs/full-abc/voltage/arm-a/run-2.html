what it does
GPU compute

No GPU instances to reserve. No hours-long setup. Specify the GPU you need as a string argument. Voltage provisions it in under 200ms, attaches it to your function, and bills to the second.

Submit a list of inputs. Voltage fans them across thousands of parallel containers, collects results, returns them in order. No queue infrastructure. No autoscaling policy. No retry logic to write.
