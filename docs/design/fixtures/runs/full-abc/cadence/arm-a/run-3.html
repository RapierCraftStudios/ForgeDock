REST API + SDKs

One endpoint, five official SDKs

Send, schedule, and query email from any stack. Full TypeScript types. Python dataclasses. Idempotency keys prevent duplicate sends. OpenAPI spec ships with every version.

02 GET /v1/emails/:id — status
03 GET /v1/analytics/templates
04 POST /v1/webhooks — events
