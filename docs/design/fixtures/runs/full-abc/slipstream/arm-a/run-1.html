How it works

01 — Connect

Your repo is the control plane.

Link your GitHub, GitLab, or Bitbucket repo. Slipstream watches every branch. Nothing to configure — the git workflow you already have is the deployment workflow.

02 — Build

Framework-native builds, zero config.

Next.js, Remix, SvelteKit, Astro, Nuxt — detected automatically. Build output is optimized, split, and prepared for the edge without a single config file.

03 — Ship

Global edge in under a second.

Assets propagate to 300+ edge locations. Functions run at the closest node. Your app is fast everywhere, not just in us-east-1.

Preview environments

Every PR is a working app.

Open a pull request and a full deployment is created automatically. Share the link with your team, client, or yourself. It's exactly what will go to production — because it's built the same way.
