Platform capabilities

01

Git-triggered builds

Push to any branch. Every commit triggers an isolated build in a clean environment. Main branch goes to production; everything else gets a preview URL.

02

Preview environments

Every PR and branch gets a live, isolated deployment. URLs are posted as PR comments. Environments are deleted when branches are merged or closed.

03

Edge functions

Write serverless functions that run at the closest edge node to each user. Sub-10ms cold starts. Web standard APIs. No container images, no provisioning.

04

Framework detection

Slipstream reads your package.json and detects Next.js, Remix, SvelteKit, Astro, Nuxt, and Gatsby automatically. Build commands, output directories, and routing are configured without you touching a setting.

05

Global CDN

Static assets are cached and served from 300+ PoPs globally. Smart cache invalidation on every deploy ensures users always get the latest build with no stale content.

06

Team collaboration

Projects, environments, and environment variables are shared across your team. Role-based access control. Deploy notifications via Slack or webhook. Full audit log.
