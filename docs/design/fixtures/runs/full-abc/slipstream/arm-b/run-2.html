Setup in 3 steps

From zero to production
in under 2 minutes.

1

Connect your repository

Authorize Slipstream on GitHub, GitLab, or Bitbucket. Select a repo. Done.

2

Configure (or don't)

We detect your framework and set build settings automatically. Override anything you want.

3

Push and ship

Every push triggers a build and deploy. Every PR gets a preview URL. Production stays separate from previews.

slipstream.yml
# Optional config — most projects need none.
name: my-app
framework: nextjs # auto-detected
build:
command: npm run build
output: .next
env:
DATABASE_URL: $DATABASE_URL
NEXT_PUBLIC_API: $API_URL
# That's everything. Push to deploy.
