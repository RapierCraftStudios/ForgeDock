import voltage
from transformers import Trainer, TrainingArguments


@voltage.function(
    gpu="A100-80GB",
    memory="120GB",
    timeout=7200,
    volumes=[voltage.Volume("model-weights")],
)
def fine_tune_llm(
    base_model: str,
    dataset: str,
    epochs: int = 3,
) -> str:
    """Fine-tune a causal LM on custom data."""
    trainer = Trainer(
        model=load_model(base_model),
        args=TrainingArguments(
            output_dir="/vol/checkpoints",
            num_train_epochs=epochs,
            fp16=True,
        ),
        train_dataset=load_dataset(dataset),
    )
    trainer.train()
    return "/vol/checkpoints/final"


# Kick off from your laptop — no cluster needed

)

How it works

3 steps
1

Decorate

Add @voltage.function to any Python function. Declare GPU type, memory, and timeout. Voltage reads your requirements.txt and builds the container automatically.

No Dockerfile required
2

Call remotely

Replace fn() with fn.remote(). Voltage cold-starts a container with your requested GPU, ships your code, runs it, and returns the result. Logs stream back in real time.

Under 3s cold start
3

Scale automatically

Map over thousands of inputs and Voltage launches thousands of parallel containers. When they finish, everything scales to zero. You pay per active compute second — nothing while idle.

0 → 1000 containers
Parallel Execution

Voltage's voltage.map() fans work out across as many parallel containers as you need. Partial results stream back as they complete. Built-in retry handles transient failures silently.
