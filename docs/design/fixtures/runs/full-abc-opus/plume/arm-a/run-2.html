Everything a software team needs. Nothing it doesn't.

Plume is opinionated about workflow so your team can focus on building. Issues, cycles, projects, roadmaps, and integrations — in a single, fast workspace that respects your time.

I
Issues
The atomic unit of work. Each issue carries a status, priority, assignee, labels, and a rich description. Create one in under a second with a single keystroke. Update everything inline — no modals, no save buttons.
Sub-issues for decomposition. Markdown with syntax highlighting. Image paste. Custom fields per workspace. Bulk operations on any filtered set.
II
Cycles
Time-boxed iterations that give your team rhythm. Drag issues in, track progress against scope, review what shipped. Unfinished work rolls forward automatically. No stand-up ceremony required.
Configurable duration: 1, 2, or 4 weeks. Burndown charts. Velocity tracking across cycles. Scope-change protection that flags mid-sprint additions.
III
Projects
Long-running initiatives that span multiple cycles. Set milestones, track completion percentage, and see exactly which sprints contributed. Projects give you the strategic view without a separate planning tool.
Roadmap timeline. Status updates with rich text. Automatic progress aggregation from child issues. Project-scoped filters and views.
IV
Integrations
Plume connects to GitHub, GitLab, Slack, Figma, and Sentry. PRs link to issues automatically by branch name. Statuses transition on merge. Errors become issues without manual triage.
GitHub: bi-directional PR/commit sync. Slack: create issues from messages, slash commands. Sentry: auto-create issues, close on deploy. Figma: design previews inline.
V
Views & Filters
Board, list, and timeline. Filter by any combination of status, priority, assignee, label, cycle, or project. Save views for your team or just yourself. Every view loads instantly.
Custom saved views. Shareable filter URLs. Group by any field. Sort by priority, updated date, or custom order. Keyboard navigation through all views.
VI
Speed
Plume is engineered to be fast. The interface responds in under 50ms. Navigation is instant. Search results appear as you type. We obsess over perceived performance because your attention matters.
Optimistic UI updates. Local-first data layer. Incremental sync. Sub-second page loads on any connection. No spinners, no loading states that linger.
