Whether you're charging a card or splitting marketplace revenue across continents, the integration looks the same.

Accept payments everywhere customers are

A single endpoint handles cards, wallets, bank transfers, and local payment methods. Tender routes each transaction to the optimal processor and handles retries, 3DS, and currency conversion automatically.

  • Smart routing increases auth rates by 8-12%
  • Automatic 3D Secure with risk-based exemptions
  • Present prices in 135 local currencies
  • Pre-built checkout UI or fully custom via Elements SDK
// Charge a customer
const payment = await tender.payments.create({
  amount: 9900,
  currency: 'eur',
  payment_method: 'pm_card_visa',
  customer: 'cus_Lk92mN',
  idempotency_key: 'ord_28491',
});

Billing that scales with your pricing model

Flat-rate, per-seat, metered, or any combination. Define pricing in the API or dashboard, and Tender calculates proration, generates invoices, retries failed payments, and sends receipts.

  • Usage-based billing with real-time metering API
  • Automatic proration on plan changes
  • Smart dunning reduces involuntary churn by 25%
  • Revenue recognition-ready invoice data
// Create a metered subscription
const sub = await tender.subscriptions.create({
  customer: 'cus_Lk92mN',
  items: [{
    price: 'price_api_calls',
    billing_scheme: 'metered',
  }],
  billing_cycle_anchor: 'now',
});

// Report usage
await tender.usage.record({
  subscription_item: sub.items[0].id,
  quantity: 14200,
});

Payouts to anyone, anywhere

Send funds to marketplace sellers, gig workers, or SaaS affiliates. Tender manages KYC, tax form collection, currency conversion, and compliance across 47 countries.

  • Automatic revenue splitting at payment time
  • 1099 / tax form generation for US recipients
  • Configurable payout schedules (daily, weekly, manual)
  • Local settlement in 28 currencies
// Split a payment between platform and seller
const payment = await tender.payments.create({
  amount: 10000,
  currency: 'usd',
  payment_method: 'pm_card_visa',
  transfer_data: {
    destination: 'acct_seller_8kL2',
    amount: 8500,  // seller gets $85
  },
  // platform keeps $15
});
