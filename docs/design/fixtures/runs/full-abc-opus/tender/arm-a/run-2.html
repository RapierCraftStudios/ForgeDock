What Tender replaces
01

Payment gateway + processor

Accept cards, wallets, bank transfers, and 50+ local methods through one API. Intelligent routing optimizes for authorization rate and cost. Automatic 3D Secure for SCA markets. Retry logic that recovers revenue silently.

02

Billing and invoicing engine

Flat-rate, per-seat, tiered, and metered models. Proration to the second on plan changes. Automated dunning with configurable retry cadence. Tax-ready invoices with line-item detail. Revenue recognition schedules for your controller.

03

Fraud and compliance stack

ML-powered risk scoring on every transaction. Custom rules engine for business-specific signals. PCI DSS Level 1 and SOC 2 Type II certified. PSD2 SCA compliance handled automatically. You never touch raw card data.

# Create a subscription with metered billing
import tender

subscription = tender.Subscription.create(
    customer="cus_acme_corp",
    items=[{
        "price": "price_pro_monthly",
        "quantity": 5,  # 5 seats
    }, {
        "price": "price_api_metered",
        # Usage reported separately
    }],
    payment_behavior="default_incomplete",
    proration_behavior="create_prorations",
    expand=["latest_invoice.payment_intent"],
)

Platform
