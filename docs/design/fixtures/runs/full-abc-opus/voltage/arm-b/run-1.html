Now available: H100 & A100 GPUs on demand

GPU compute that feels like local Python

Write a function. Add a decorator. Voltage runs it on cloud GPUs, scales to thousands of containers, and bills you only while code is executing. No clusters. No YAML. No idle costs.

Run your first function Read the docs
inference.py
import voltage

@voltage.function(
    gpu="A100-80GB",
    memory=32768,
    timeout=300,
)
def generate_embeddings(texts: list[str]):
    # Runs on a cloud A100 — not your laptop
    from sentence_transformers import SentenceTransformer
    model = SentenceTransformer("all-MiniLM-L6-v2")
    return model.encode(texts).tolist()

# Call it like any Python function
result = generate_embeddings(["hello world"])
print(result) # [[0.012, -0.034, ...]]

Three steps to cloud compute

01

Install the SDK

One pip install gives you the Voltage decorator and CLI. Works with any Python 3.9+ environment.

pip install voltage
02

Decorate your function

Specify GPU type, memory, and timeout. Your function's dependencies are detected automatically from your imports.

@voltage.function(gpu="A100")
03

Run it

Call your function normally or deploy it as an endpoint. Voltage provisions the hardware, runs the code, and tears it down.

voltage deploy inference.py
