import voltage

# Define your compute environment once
app = voltage.App("image-gen")

image = voltage.Image.debian_slim()
  .pip_install("torch", "diffusers", "transformers")

@app.function(
    gpu="A100-80GB",
    image=image,
    timeout=600,
)
def generate(prompt: str) -> bytes:
    from diffusers import StableDiffusionPipeline
    pipe = StableDiffusionPipeline.from_pretrained(
        "stabilityai/sdxl-turbo",
    )
    result = pipe(prompt).images[0]
    return result.tobytes()

# Deploy as an API endpoint
@app.endpoint(method="POST")
def api(prompt: str):
    return generate.remote(prompt)

Built for compute-heavy Python

01 / GPU

On-demand GPUs

A100, H100, L4 — attach any GPU with a single parameter. No provisioning, no reservations. Billed per second of actual use.

02 / Scale

Zero to 10,000

Containers scale automatically based on demand. Burst to thousands for batch jobs, scale back to zero between requests. No capacity planning.

03 / Deploy

Decorator-based deploys

No Dockerfiles, no Kubernetes YAML, no Terraform. Add @app.function() to your Python and deploy. Your local code runs in the cloud.

04 / Endpoints

Inference endpoints

Serve any model as an auto-scaling HTTP endpoint. Built-in request batching, GPU warmup pools, and automatic health checks.

05 / Schedule

Cron & pipelines

Schedule training runs, ETL jobs, or batch inference. Chain functions into DAG pipelines with automatic retry and monitoring.

06 / Dev

Local-feeling loop

Interactive debugging with voltage shell. Hot-reload code changes. Stream logs in real time. The cloud feels like your laptop.
