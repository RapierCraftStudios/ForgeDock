inference.py
import voltage
 
app = voltage.App("llm-service")
 
# Container image with dependencies baked in
image = (
  voltage.Image.debian_slim(python="3.11")
  .pip_install("vllm", "torch")
)
 
@app.cls(gpu="H100", image=image)
class LLM:
  @voltage.enter()
  def load(self):
    from vllm import LLM
    self.engine = LLM("meta-llama/Llama-3-70B")
 
  @voltage.method()
  def generate(self, prompt: str) -> str:
    output = self.engine.generate(prompt)
    return output[0].outputs[0].text
 
@app.endpoint(method="POST")
def serve(prompt: str):
  return LLM().generate.remote(prompt)

How it works

Write Python.
Add a decorator.
Deploy.

Voltage serializes your function, builds a container image with your dependencies, and deploys it to our global fleet. When a request comes in, we boot a container in under 300ms, run your code on the GPU you specified, and shut it down when done. You only pay for the seconds your code is actually running.
