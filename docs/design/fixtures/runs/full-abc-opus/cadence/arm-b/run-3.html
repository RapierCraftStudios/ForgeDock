Deliverability

Your email lands in the inbox. We make sure of it.

Deliverability is the hardest part of email and the most invisible. Cadence manages domain authentication, IP reputation, bounce processing, and suppression lists automatically.

  • SPF, DKIM, and DMARC configured with one DNS change
  • Dedicated IP pools with automated warm-up
  • Real-time bounce and complaint suppression
  • Blocklist monitoring and proactive alerts
  • ISP postmaster integration for Gmail, Outlook, Yahoo

Testing

Test every email before it reaches a real inbox

Cadence's test mode captures outgoing email in a virtual inbox. Your integration tests can assert on email content, and your team can preview rendered templates with live data.

  • Sandbox mode: all emails captured, none delivered
  • API to query captured emails in test suites
  • Visual preview with real template data
  • Separate test and production API keys
// Test mode: emails are captured, not sent
const cadence = new Cadence('cd_test_...');

await cadence.send({
  to: 'test@example.com',
  template: 'welcome',
  data: { name: 'Test User' }
});

// Assert in your test suite
const inbox = await cadence.testInbox('test@example.com');
expect(inbox[0].subject).toContain('Welcome');
expect(inbox[0].html).toContain('Test User');
