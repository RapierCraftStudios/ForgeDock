For engineers

Integrate in minutes

Install the SDK, set your API key, and call cadence.send(). Templates are referenced by slug, so you never hardcode HTML in application code.

npm install @cadence/sdk

# Send a test email
npx cadence send \
  --to test@example.com \
  --template welcome-email \
  --data '{"name": "Test"}'
