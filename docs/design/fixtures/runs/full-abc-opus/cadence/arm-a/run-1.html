01 The API

Three lines to send. One endpoint to learn.

The core API is one POST to /v1/emails/send. Pass a recipient, a template ID (or inline HTML), and your data. Cadence handles DKIM signing, retry logic, bounce processing, and delivery tracking. SDKs for Node, Python, Go, Ruby, and PHP.

Node.js 
import { Cadence } from '@cadence/node';

const cadence = new Cadence('cd_live_9x8f...');

await cadence.send({
  from:     'notifications@yourapp.com',
  to:       'jane@customer.io',
  template: 'welcome-series-1',
  data:     { first_name: 'Jane', plan: 'Pro' },
});
// => { id: 'msg_8xK2p', status: 'queued' }
Python 
from cadence import Cadence

client = Cadence(api_key="cd_live_9x8f...")

response = client.send(
    from_email="billing@yourapp.com",
    to="jane@customer.io",
    template="invoice-receipt",
    data={"amount": "$49.00", "invoice_id": "INV-1042"},
)
# response.id => "msg_r3Wp1"
02

Built for production

Cadence runs on dedicated IPs with automatic warm-up, handles suppression lists, processes bounces and complaints in real time, and provides domain health monitoring. This is infrastructure, not a weekend project.

  • Automatic IP warm-up — graduated volume increase over 14 days
  • Suppression management — global + account-level suppression lists
  • Bounce classification — hard/soft bounce detection with auto-suppression
  • Complaint loops — ISP feedback loop integration (AOL, Yahoo, Outlook)
  • Webhook events — delivery, open, click, bounce, complaint, unsubscribe
  • Rate limiting — per-domain throttling to protect sender reputation
03

Domain authentication

One API call generates your SPF, DKIM, and DMARC records. Cadence continuously monitors DNS propagation and alerts you if records drift or expire. BIMI support included.

05 Templates

Write templates in code. Edit them in a visual tool.

Engineers define email templates as React-like components. Content owners edit text, images, and variables in a visual editor without touching the codebase. Version-controlled, preview-able, and deployable via API or dashboard.

Template JSX 
import { Email, Section, Button }
  from '@cadence/components';

export default function WelcomeEmail({ first_name, plan }) {
  return (
    <Email subject={`Welcome, ${first_name}`}>
      <Section padding="32">
        <h1>You're on {plan}</h1>
        <p>Hi {first_name}, your account
           is ready.</p>
        <Button href="/dashboard">
          Open Dashboard
        </Button>
      </Section>
    </Email>
  );
}
