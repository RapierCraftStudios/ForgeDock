Quick Start
Node.js Send 
import { Cadence } from '@cadence/node';

const cadence = new Cadence('cd_live_9x8f...');

await cadence.send({
  from:     'team@yourapp.com',
  to:       'jane@customer.io',
  template: 'welcome-v2',
  data:     {
    first_name: 'Jane',
    plan:       'Pro',
  },
});
// => { id: 'msg_8xK2p', status: 'queued' }

Core

Transactional Send

Single POST, single recipient. Template or inline HTML. Attachments, inline images, custom headers, reply-to. Queued and delivered in under 2 seconds.

Scale

Batch Send

Up to 10,000 recipients per call with per-recipient data merge. Schedule for later or send immediately. Progress tracked via webhook events.

Content

Template Engine

React-style components for engineers. Visual drag-and-drop editor for content owners. Version history, preview with test data, deploy via API or dashboard.

Auth

Domain Setup

One API call generates SPF, DKIM, and DMARC records. Continuous DNS monitoring with alerts on drift. BIMI support for brand logos in inbox.

Safety

Suppression Lists

Global and per-account suppression. Auto-add on hard bounce, complaint, or unsubscribe. CSV import/export. Never send to a bad address twice.

Events

Webhook Stream

Real-time events for delivery, open, click, bounce, complaint, and unsubscribe. Retry with exponential backoff. Filterable by type, template, and tag.

Templates as components

Write email templates as React-like JSX components. Import shared layouts, buttons, and sections. Your content team edits copy and images in a visual tool that compiles to the same component tree.

  • Component library — Email, Section, Button, Text, Image, Divider, Spacer
  • Variables — Dynamic data injection with type-safe schemas
  • Conditionals — Show/hide blocks based on recipient data
  • Loops — Iterate over arrays (line items, notifications)
  • Preview — Render with test data, inspect HTML and plain text
  • Version control — Full history, rollback, branch previews
Template JSX welcome-v2 
import { Email, Section, Button, Text, Image }
  from '@cadence/components';

export default function WelcomeEmail({
  first_name,
  plan,
  login_url,
  features,
}) {
  return (
    <Email
      subject={`Welcome, ${first_name}`}
      preview="Your account is ready"
    >
      <Section padding="32" bg="#ffffff">
        <Image
          src="https://cdn.yourapp.com/logo.png"
          width=120
          alt="YourApp"
        />
        <Text size="xl" weight="bold" mt="24">
          Welcome aboard, {first_name}
        </Text>
        <Text color="muted" mt="12">
          You're on the {plan} plan.
        </Text>

        {features.map(f => (
          <Text key={f} size="sm">
            {f}
          </Text>
        ))}

        <Button
          href={login_url}
          bg="#FF3D00"
          mt="24"
        >
          Open Your Dashboard
        </Button>
      </Section>
    </Email>
  );
}
