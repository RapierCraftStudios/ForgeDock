01

Connect

Link your GitHub, GitLab, or Bitbucket repository. Slipstream reads your framework and configures the build pipeline.

$ slipstream link github.com/team/app
02

Push

Push to any branch. Slipstream builds and deploys automatically. Preview URLs for branches, production URL for main.

$ git push origin feature/auth
03

Ship

Merge your PR. Production updates in seconds. Instant rollback if anything goes wrong. Global CDN invalidation included.

$ gh pr merge --squash
