How it works

3 steps
01

Connect your repo

Import from GitHub, GitLab, or Bitbucket. Slipstream detects your framework and configures the build.

$ slipstream init
02

Push to any branch

Every branch gets a preview deployment. Every pull request gets a unique URL with its own environment.

$ git push origin feature/auth
03

Merge and ship

Merge to main. Production updates in seconds. Atomic deploys with instant rollback if needed.

$ gh pr merge --squash
