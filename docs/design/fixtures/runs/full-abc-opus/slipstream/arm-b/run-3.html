serverless functions

Backend logic without the backend

Write API endpoints as files. They deploy alongside your frontend to the same edge network. Automatic scaling. No cold starts. No instance types. No load balancers. You write the function; Slipstream handles the rest.

// api/users/[id].ts
import { db } from '@/lib/db'
 
export async function GET(req) {
  const user = await db.user
    .findUnique({ id: req.params.id })
 
  return Response.json(user)
}
